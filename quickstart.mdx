## 1. Get your API key

**Platform API Keys**, and create a new key. You'll see the full key **once** —
copy it somewhere safe immediately.

## 2. Make a test request

Verify your key by calling the `platforms/me` endpoint:

<CodeGroup>

```bash cURL
curl https://byveya.com/api/v1/platforms/me \
  -H "x-api-key: bv_live_YOUR_KEY_HERE"
```

```javascript Node.js
const res = await fetch("https://byveya.com/api/v1/platforms/me", {
  headers: { "x-api-key": "bv_live_YOUR_KEY_HERE" },
});
const data = await res.json();
console.log(data);
```

```python Python
import httpx

r = httpx.get(
    "https://byveya.com/api/v1/platforms/me",
    headers={"x-api-key": "bv_live_YOUR_KEY_HERE"},

print(r.json())
```

## 3. Send your first SMS

```bash
curl -X POST https://byveya.com/api/v1/sms/send \
  -H "x-api-key: bv_live_YOUR_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+447911123456",
    "message": "Hello from ByVeya! 🚀"
  }'
```

## 4. Schedule a social post

```bash
curl -X POST https://byveya.com/api/v1/social/posts \
  -H "x-api-key: bv_live_YOUR_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "network": "instagram",
    "caption": "Exciting news coming soon! 🔥",
    "hashtags": ["announcement", "byveya"],
    "scheduled_at": "2026-04-01T09:00:00Z"
  }'
```
