# ByVeya Platform API

The ByVeya Platform API gives your applications direct access to the full suite
of ByVeya tools through a single, versioned REST interface. Whether you're
connecting an internal project or an external SaaS platform, every feature is
available through the same authenticated endpoint.

## Base URL

All API requests are made to:

```
https://byveya.com/api/v1
```

## Versioning

The API is versioned via the URL path (`/api/v1`). Breaking changes will
always be released under a new version. The current version will remain
supported for a minimum of 12 months after a new major version is published.
